# AI Story Weaver 项目总览

## 📋 项目简介

**项目名称**: AI Story Weaver (AI故事编织者)
**项目类型**: 沉浸式AI互动故事小说App
**基于项目**: AiChat Android应用
**目标用户**: 喜欢互动故事和AI创作的移动用户

## 🎯 核心特色

### 1. 多类型故事模板系统
- **5+故事类型**: 冒险、浪漫、悬疑、奇幻、科幻
- **难度分级**: 简单、中等、困难
- **自定义编辑器**: 可视化故事节点编辑
- **预设模板库**: 丰富的开箱即用模板

### 2. 智能分支选择引擎
- **条件判断系统**: 基于玩家属性的动态选择
- **权重计算算法**: 个性化推荐最优路径
- **路径追踪机制**: 完整记录选择历史
- **复杂逻辑支持**: AND/OR条件组合

### 3. 多结局支持
- **多种结局类型**: 好结局、坏结局、隐藏结局
- **成就解锁系统**: 基于选择的成就收集
- **统计数据分析**: 玩家行为模式分析
- **重玩价值**: 多路径探索体验

### 4. 沉浸式阅读体验
- **打字机效果**: 逐字显示的故事文本
- **主题定制**: 深色/浅色模式切换
- **动画过渡**: 流畅的页面切换效果
- **音效支持**: 背景音乐和环境音

## 🏗️ 技术架构

### 前端架构
```
┌─────────────────────────────────────────┐
│              Presentation Layer         │
│  (Jetpack Compose UI + ViewModels)      │
├─────────────────────────────────────────┤
│               Domain Layer              │
│  (Use Cases, Repositories, Entities)    │
├─────────────────────────────────────────┤
│               Data Layer                │
│  (Local DB + Remote APIs + Caching)     │
└─────────────────────────────────────────┘
```

### 技术栈
- **语言**: Kotlin
- **框架**: Jetpack Compose, MVVM
- **数据库**: Room, DataStore
- **网络**: Retrofit, OkHttp
- **依赖注入**: Hilt
- **状态管理**: StateFlow, SharedFlow
