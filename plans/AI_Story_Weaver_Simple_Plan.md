# AI Story Weaver 简化版开发计划

## 🎯 项目概述

**项目名称**: AI Story Weaver (简化版)
**项目目标**: 基于现有AiChat应用，创建简单的AI互动故事功能
**核心需求**: 接入个人API服务商生成小说内容，无需复杂功能

## 📋 简化需求分析

### ✅ 必需功能
- [x] 基于现有AiChat应用的架构
- [x] 接入个人API服务商
- [x] 生成小说内容
- [x] 基本的互动选择
- [x] 故事进度保存

### ❌ 不需要的功能
- [x] 图像生成功能
- [x] VR/AR体验
- [x] 社交分享
- [x] 商业化功能
- [x] 复杂的成就系统
- [x] 多结局统计分析
- [x] 高级UI动画
- [x] 主题定制
- [x] 应用商店上架

## 🏗️ 简化技术方案

### 架构调整
```
现有AiChat应用
    ↓
添加简单的故事功能模块
    ↓
保持原有架构不变
    ↓
简单的故事界面和逻辑
    ↓
发布为个人使用应用
```

### 核心数据模型 - 简化版
```kotlin
// 简化的故事模板
data class SimpleStoryTemplate(
    val id: Long,
    val title: String,
    val genre: String, // "冒险", "浪漫", "悬疑"等
    val systemPrompt: String,
    val initialScene: String
)

// 简化的故事节点
data class StoryNode(
    val id: String,
    val text: String,
    val choices: List<Choice> = emptyList()
)

// 简化的选择
data class Choice(
    val id: String,
    val text: String,
    val nextNodeId: String? = null
)

// 简化的故事进度
data class StoryProgress(
    val templateId: Long,
    val currentNodeId: String,
    val choicesMade: List<String> = emptyList(),
    val lastPlayedTime: Long = System.currentTimeMillis()
)
```
