# AI Story Weaver - 快速开始指南

## 🚀 3分钟上手

### 第一步：配置API (1分钟)
1. 打开应用，点击右上角的"设置"图标
2. 在"API Key"字段输入你的OpenAI API密钥
3. 保存设置

### 第二步：创建第一个故事 (1分钟)
1. 点击主界面中的"AI故事创作"按钮
2. 选择"奇幻冒险之旅"模板
3. 设置难度为"新手"
4. 填写角色设定：勇敢的年轻法师
5. 点击"生成大纲"

### 第三步：阅读和体验 (1分钟)
1. 查看生成的故事大纲
2. 点击"开始创作"
3. 按章节顺序阅读故事
4. 在第8章时体验分支选择

## 📋 详细步骤

### 配置API密钥
```bash
# 获取OpenAI API密钥
# https://platform.openai.com/api-keys

# 在应用中配置
1. 进入"设置"页面
2. 找到"API Key"输入框
3. 粘贴你的密钥：sk-xxxxxxxxxxxxxxxxxxxx
4. 点击保存
```

### 创建故事
```
1. 点击"AI故事创作"
2. 选择故事类型：
   - 奇幻冒险之旅
   - 未来科幻世界
   - 悬疑推理案件
   - 古风武侠传奇
   - 现代爱情故事
   - 无限流生存挑战
3. 选择难度等级：
   - 新手：简单直接的故事发展
   - 进阶：适度挑战，需要思考
   - 专家：复杂情节，多重分支
4. 自定义设定：
   - 角色描述：主角的性格、外貌、技能
   - 背景描述：故事发生的世界、时代
   - 特殊要求：任何特殊的创作需求
5. 点击"生成大纲"
6. 预览大纲内容
7. 点击"开始创作"
```

### 阅读故事
```
1. 按章节顺序阅读
2. 观察进度指示器（顶部）
3. 在分支点做出选择
4. 体验不同的故事走向
5. 随时可以重新开始
```

## 🎯 核心概念

### 故事模板
每个模板都有预设的系统提示词，确保故事风格一致：
- **奇幻冒险**：魔法世界，神秘传说
- **科幻**：未来科技，星际旅行
- **悬疑推理**：案件调查，逻辑推理
- **古风武侠**：江湖恩怨，武功对决
- **现代言情**：都市情感，成长经历
- **无限流**：恐怖求生，策略思考

### 难度等级
- **新手**：线性故事，较少选择，适合初次体验
- **进阶**：适度分支，需要一些思考决策
- **专家**：复杂情节，多重分支，高难度挑战

### 分支系统
- 每8-10章出现一个关键分支点
- 每个分支提供3-4个合理的选择
- 选择会影响后续故事发展
- 最终到达不同的故事结局

## 💡 使用技巧

### 角色设定建议
```
好的角色描述应该包含：
- 基本信息：年龄、性别、外貌特征
- 性格特点：勇敢、聪明、善良等
- 特殊技能：魔法、格斗、编程等
- 背景故事：出身、经历、目标
```

### 背景描述建议
```
好的背景设定应该包含：
- 世界观：时间、地点、社会结构
- 环境特征：气候、地理、文化
- 重要元素：魔法系统、科技水平等
- 冲突来源：主要矛盾和挑战
```

### 特殊要求示例
```
- "主角必须保持乐观积极的态度"
- "加入更多战斗场面"
- "强调团队合作的重要性"
- "增加悬疑元素"
```

## 🔧 故障排除

### API连接问题
```
错误："API调用失败"
解决方案：
1. 检查API密钥是否正确
2. 确认网络连接正常
3. 查看API余额是否充足
4. 尝试重新生成故事
```

### 加载缓慢
```
如果故事生成很慢：
1. 检查网络速度
2. 减少角色设定的复杂度
3. 简化背景描述
4. 稍后再试
```

### 故事质量不佳
```
如果故事不符合预期：
1. 重新调整角色设定
2. 修改背景描述
3. 添加更具体的特殊要求
4. 选择不同的故事模板
```
